# Interview Intelligence Dossiers

Auto-generated when a recruiter response is detected from a pipeline company.

Each dossier contains:
1. Company overview (recent news, tech stack, digital maturity)
2. Key leadership (who Ahmed would report to)
3. Strongest talking points for that specific role
4. Salary benchmarks for that title in that geography
5. 3 questions to ask that show homework was done

Triggered by: email cron recruiter response detection -> .trigger file -> dossier sub-agent
