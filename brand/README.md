# Brand Assets

Reusable Ahmed Nasr brand and visual-generation guidance for LinkedIn, executive visuals, and AI-image prompt workflows.

## Contents

- `ahmed-master-portrait-generator-prompt.md` — master prompt for consistent executive portrait generation.
- `ahmed-visual-library-v1.md` — reusable visual library and build-order guidance.
- `gpt-image-2-brand-system.md` — brand visual system, lanes, workflow, and guardrails.
- `gpt-image-2-compact-prompt-template.md` — compact prompt template for fast visual generation.
- `gpt-image-2-four-lane-prompt-pack.md` — four-lane prompt pack for common visual directions.
- `linkedin-visual-brief-template.md` — brief template for LinkedIn visual planning.

## Ownership

Owned by the workspace brand/content automation flow. Keep this directory tracked so CMO/CTO agents can reuse it and so heartbeat checks do not flag it as a root-level untracked leak.

## Safety rule

Do not store private access material, client-confidential data, or sensitive unpublished personal material here.
