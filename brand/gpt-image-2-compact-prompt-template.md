# GPT Image 2.0 - Compact Prompt Template

Use this as the default one-shot prompt template.

```text
Create a premium visual of Ahmed Nasr, a senior executive in AI-enabled execution and transformation leadership.

Goal:
[authority post / scaling post / AI strategy post / statement visual]

Visual lane:
[Lane A Executive Editorial / Lane B Operator in Motion / Lane C AI Strategy / Lane D Statement Visual]

Scene:
[modern executive office / command center / strategy room / elegant corporate environment]

Mood:
[calm / strategic / premium / cinematic / high-trust / composed under pressure]

Composition:
[portrait / half-body / walking shot / left-third composition / centered / negative space on right]

Wardrobe:
[tailored dark suit / crisp white shirt / executive business attire]

Lighting:
[cinematic soft side light / premium editorial lighting / subtle contrast / realistic skin texture]

Requirements:
- strong facial likeness to Ahmed Nasr
- realistic hands
- believable executive posture
- premium magazine-style realism
- elegant and credible business environment
- restrained, sophisticated color grading
- leave clean negative space for later typography if needed

Avoid:
- fake text
- distorted hands
- plastic skin
- cheesy grin
- cluttered dashboard backgrounds
- hologram clichés
- exaggerated sci-fi effects
- startup-bro vibe
- cheap motivational poster feel
- unrealistic suit details

The result should feel authoritative, strategic, modern, and premium.
```

## Refinement follow-up prompt

```text
Keep the same concept and composition. Improve Ahmed’s facial likeness, clean the hands, refine the suit and posture, make the lighting more premium and editorial, simplify the background, preserve negative space, and make the image feel more credible, realistic, and high-trust.
```
