## Canva Specific Instructions

1. Log in to Canva (export_website)
2. Open website editor
3. Update each page:
   - Home page
   - What We Do
   - About
   - Contact
4. Use "Pages" sidebar to navigate
5. Click elements to edit text
6. Use "Publish" button to save changes
7. Preview before publishing

### Design Tips for Healthcare
- Colors: Blue (#0056b3), Green (#28a745), White
- Font: Clean sans-serif (Montserrat, Open Sans)
- Icons: Use medical/healthcare icons
- Images: Hospital, technology, team photos
- Spacing: Clean, plenty of white space
