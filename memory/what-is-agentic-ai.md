## Definition

> "A system built around an AI model that enables it to operate more like an active teammate, capable of setting goals, taking action, and working independently."

| Component | What It Does |
|-----------|-------------|
| Memory | Retains context |
| Tools | Access to codebase, terminal, etc. |
| Goal | Defined objective |
| Autonomy | Acts independently |

---

## The 5-Step Process

| Step | What Happens |
|------|--------------|
| **1. Receives goal** | Natural language prompt, clear goals help agent decide |
| **2. Plans tasks** | Breaks goal into steps |
| **3. Uses tools + context** | Connects to systems, uses memory |
| **4. Takes action** | Carries out plan until verified complete |
| **5. Learns + adapts** | Feedback loop for better decisions |

---

## Agentic AI vs Generative AI

| Generative AI | Agentic AI |
|--------------|------------|
| Creates content | Acts on goals |
| Done after output | Continues to iterate |
| Passive | Active |

---

## Key Benefits

| Benefit | Description |
|---------|-------------|
| **Efficiency + Scale** | Manages repetitive tasks with minimal supervision |
| **Autonomous Workflows** | Adapts to changing inputs, coordinates multiple steps |
| **Real-time Response** | Adjusts if priorities shift or new data appears |
| **Focus on High-value** | Frees people for strategic work |

---

## Key Risks

| Risk | Description |
|------|-------------|
| **Prompt Injection** | Malicious prompts override original instructions |
| **Sensitive Data Leaks** | Unintentional sharing of sensitive data |
| **Supply Chain Vulnerabilities** | New entry points for attackers |
| **Bias** | Amplified if not monitored |
| **Misalignment** | Goals drift from human intent |
| **API Misuse** | Unauthorized access through API connections |
