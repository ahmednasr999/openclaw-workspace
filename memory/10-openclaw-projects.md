## Quick Reference: Skill Vetting Checklist

Before installing ANY community skill:

- [ ] Read SKILL.md completely
- [ ] Check publisher verification status
- [ ] Review permission requests
- [ ] Search ClawHub discussions for security reports
- [ ] Test in isolated environment first
- [ ] If requires external downloads: DO NOT INSTALL

---

## Red Flags

| Flag | Meaning |
|------|----------|
| Asks for camera permission | SCAM |
| Asks for filesystem permission | Verify carefully |
| External downloads | DON'T INSTALL |
| No source code review | Skip |
