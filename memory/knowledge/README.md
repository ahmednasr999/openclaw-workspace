# Knowledge Bank

**Purpose:** Structured storage for all external content Ahmed shares (articles, tweets, videos, frameworks, documents). Indexed by QMD for semantic retrieval.

**Rules (ALL models, ALL sessions):**

1. When Ahmed shares any content (URL, screenshot, copy-paste, file), ALWAYS:
   - Analyze it
   - Extract key insights
   - Save to the correct file below using the standard template
   - Confirm: "Saved to knowledge bank: [filename]"

2. NEVER just analyze and forget. If it was worth analyzing, it's worth saving.

3. When drafting LinkedIn posts, preparing for interviews, or doing any strategic work, ALWAYS search the knowledge bank first.

## Retrieval Rules

- Before drafting any LinkedIn post: search knowledge/content-strategy.md + knowledge/raw-inspiration.md
- Before interview prep: search knowledge/leadership-insights.md + knowledge/industry-trends.md
- Before any strategic decision: search all knowledge files
- When connecting dots: pull from multiple knowledge files and cross-reference
