### LinkedIn Comment API - Key Technical Learning
- **`LINKEDIN_CREATE_COMMENT_ON_POST` (Composio tool):** Does NOT work - returns "not found" for both `urn:li:ugcPost` and `urn:li:share` formats
- **Working method:** `proxy_execute("POST", "/v2/socialActions/{encoded_urn}/comments", "linkedin")`
- **URN gotcha:** Activity IDs from post URLs don't always match ugcPost IDs. Some posts work with `urn:li:activity:{id}`, others return HTTP 400 with the correct `urn:li:ugcPost:{actual_id}` in the error message
- **Retry strategy:** Post with activity URN first; if 400, extract correct ugcPost URN from error and retry
- **Rate limiting:** 3 concurrent comments works fine; added 1-2 second delays between batches
