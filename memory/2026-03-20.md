### Cron Registration Syntax (Learned)
- `openclaw cron add` uses `--cron "<expr>"` not `--schedule`
- `--message "exec: python3 /path/to/script.py"` for script execution
- `--timeout-seconds <n>` for timeout
- `--no-deliver` to suppress chat announcements
- `--light-context` for minimal bootstrap
