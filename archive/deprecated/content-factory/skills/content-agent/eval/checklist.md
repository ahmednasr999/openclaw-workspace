# Content Quality Gate — Binary Pass/Fail

Every item must pass. One failure = revision required. Max 2 revision cycles before escalating to Ahmed.

## Hook Quality
- [ ] Hook would make a GCC CTO stop scrolling
- [ ] Hook is under 15 words
- [ ] Hook does not start with "I'm excited..." or any off-voice opener
- [ ] The insight/payoff comes AFTER the "see more" fold

## Voice & Tone
- [ ] Sounds like Ahmed, not a brand or content mill
- [ ] Contains at least one specific detail (numbers, names, dates, countries)
- [ ] No buzzwords (synergy, leverage, ecosystem, disrupt, game-changer)
- [ ] No em dashes anywhere
- [ ] Uses contractions naturally
- [ ] Does not mention "Saudi German Hospital Group" by name

## Structure & Format
- [ ] Short paragraphs with line breaks (1-2 sentences per paragraph)
- [ ] Under 1300 characters (2000 for story format)
- [ ] No links in post body
- [ ] 3-5 specific hashtags at bottom only (not generic like #business)
- [ ] Ends with a specific engagement question (not "what do you think?")

## Content Quality
- [ ] Anchored in a content pillar (PMO, DT, AI, GCC, Leadership)
- [ ] Has a defensible point of view (not neutral/generic)
- [ ] Post type matches the selected type (story structure for story, etc.)
- [ ] Information is factually accurate per master CV data
- [ ] No fabricated roles, titles, metrics, or achievements

## Strategic Checks
- [ ] Content mix: this post doesn't create >10% surplus in any type (check mix tracker)
- [ ] If experiment week: post correctly implements the test variable
- [ ] If expertise/data post: contains a named concept or framework (AI citation optimization)

## Image (if applicable)
- [ ] Image genre matches post type per mapping table
- [ ] Brand colors correct (bg #0D1421, correct accent for theme)
- [ ] No scrollbar artifacts
- [ ] Text legible at mobile size
- [ ] Footer present with correct credentials
