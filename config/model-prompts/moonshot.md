# Moonshot Model Prompt Guide (Kimi K2.5)

## Strengths
- Strong alternative reasoning engine
- Good at research and analysis
- Mid-tier cost
- Best for: second opinions, research tasks, alternative perspectives

## Prompting Style
- Similar to Anthropic style
- Benefits from clear context setting upfront
- Good with comparative analysis ("compare X vs Y")

## Avoid
- Don't use as primary model for critical tasks
- Don't expect Opus-level strategic depth
