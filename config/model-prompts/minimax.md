# MiniMax Model Prompt Guide (M2.1 / M2.5)

## Strengths
- Free tier - no cost concerns
- Fast response times
- Good at following structured instructions
- Best for: script execution, cron tasks, data formatting, simple Q&A

## Prompting Style
- Be extremely explicit - spell out every step
- Provide examples of expected output format
- Keep prompts focused on ONE task
- Use numbered steps for multi-step workflows

## Avoid
- Don't expect deep reasoning or nuanced analysis
- Don't give vague instructions ("make it better")
- Don't rely on it for creative writing or strategic thinking
- Don't use for tasks requiring judgment calls - escalate to Opus
