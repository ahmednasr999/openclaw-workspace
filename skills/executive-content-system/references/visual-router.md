# Visual Router

Use this guide when deciding what visual should accompany an executive LinkedIn post.

## Premium statement card

Use when the post has one strong claim.

Pattern:
- dark/cinematic topic-relevant background
- one line or two-line headline
- Ahmed Nasr lockup
- no dense text

## Metric card

Use when the post leads with a number.

Pattern:
- large number
- short interpretation
- source note if applicable
- restrained accent color

## Framework card

Use when the post has a model, checklist, maturity ladder, or governance system.

Pattern:
- 3-5 blocks max
- readable labels
- strong hierarchy
- no tiny captions

## Comparison card

Use when the post contrasts two models or decisions.

Pattern:
- split layout
- left/right contrast
- clear executive implication

## Carousel

Use when one image would be too dense.

Best for:
- 5-step frameworks
- transformation playbooks
- maturity models
- shift/risk/rule/action narratives

Default flow:
1. Hook
2. The shift
3. The risk
4. The operating rule
5. The practical action
6. Question or CTA

## Avoid

- meaningless robots
- abstract neon geometry
- busy backgrounds behind text
- stock-photo handshakes
- hospital/fintech clichés unless grounded in the actual post
- text smaller than mobile-readable size
