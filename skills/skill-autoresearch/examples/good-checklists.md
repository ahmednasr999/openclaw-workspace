# Example Checklists

Pre-built checklists for common skill types. Use these as starting points.

---

## CV Builder Skill

1. [ ] Does the Professional Summary mention the target role's industry?
   - PASS: Summary explicitly names the sector (e.g., "HealthTech", "FinTech")
   - FAIL: Generic summary with no sector mention

2. [ ] Are the top 3 JD keywords present in the first page?
   - PASS: At least 3 of the top 5 JD keywords appear in Summary + first role
   - FAIL: Fewer than 3

3. [ ] Does every bullet follow the AVR pattern (Action + Value + Result)?
   - PASS: Every bullet starts with action verb and includes a metric
   - FAIL: Any bullet starts with "Responsible for" or has no metric

4. [ ] Is the output free of em dashes, tables, and multi-column layouts?
   - PASS: Clean single-column, no em dashes
   - FAIL: Any ATS-breaking formatting found

5. [ ] Is the file name format correct? ("Ahmed Nasr - [Title] - [Company].pdf")
   - PASS: Exact format with spaces and dashes
   - FAIL: Underscores, missing company, wrong format

---

## LinkedIn Post Skill

1. [ ] Does the first line create a hook (pattern interrupt, bold claim, or question)?
   - PASS: First line would make you stop scrolling
   - FAIL: Generic opener ("I'm excited to share...")

2. [ ] Is every paragraph 1-2 sentences max?
   - PASS: No paragraph exceeds 2 sentences
   - FAIL: Any wall of text

3. [ ] Is the post free of buzzwords ("synergy", "leverage", "ecosystem", "disrupt")?
   - PASS: Zero buzzwords
   - FAIL: Any buzzword found

4. [ ] Does it end with a question or clear CTA?
   - PASS: Last line is a question or specific call to action
   - FAIL: Just trails off

5. [ ] Does it include at least one specific detail (number, name, date)?
   - PASS: Contains concrete specifics
   - FAIL: All generic statements

---

## Email Writer Skill

1. [ ] Is the subject line under 50 characters?
   - PASS: Under 50 chars
   - FAIL: Over 50 chars

2. [ ] Does the opening line reference something specific about the recipient?
   - PASS: Personalized opener
   - FAIL: Generic "I hope this finds you well"

3. [ ] Is there exactly one clear ask/CTA?
   - PASS: Single clear next step
   - FAIL: Multiple asks or no ask

4. [ ] Is the total email under 150 words?
   - PASS: Under 150 words
   - FAIL: Over 150 words
