# Weekly Report Starter

A concrete starter deck for recurring weekly business reviews.

Purpose:
- provide one realistic weekly-report reference built from the redesigned `slides` lane
- show how status, metrics, risks, timeline, and next steps can work in a compact reporting deck
- give the lane a second practical editable example beyond executive-summary mode

Build:

```bash
NODE_PATH=/root/.openclaw/workspace/skills/slides/node_modules node deck.js
```

Output:
- `exports/weekly-report-starter.pptx`
