# Editorial Pattern Showcase

A small editable showcase deck for the redesigned `slides` skill.

Purpose:
- give the new pattern system a concrete reference deck
- show how editorial pattern logic should affect default composition
- provide an editable PPTX source, not just screenshots or prose docs

Included patterns:
- cover
- section-divider
- agenda
- thesis-summary
- metric
- comparison
- 2-column-explainer
- process-timeline
- closing

Build:

```bash
NODE_PATH=/root/.openclaw/workspace/skills/slides/node_modules node deck.js
```

Output:
- `exports/slides-editorial-pattern-showcase.pptx`
