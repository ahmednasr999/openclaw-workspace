# Last 30 Days Review

Use this skill when Ahmed asks for a recent-period review, recap, audit, or pattern scan over roughly the last month.

## Process

1. Identify the target lane: CEO, HR, CTO, CMO, JobZoom, or all.
2. Prefer local source files first: reports, memory notes, `.learnings`, traces, and dated artifacts.
3. Summarize decisions, completed work, failures, repeated issues, and unresolved risks.
4. Highlight what should be promoted into a durable rule, skill, script, or check.
5. Keep the output executive-friendly: decisions first, evidence second.

## Safety

- Do not expose secrets or raw private data.
- For external sends or public posts, require explicit approval.
- If findings repeat across weeks, apply the NASR Skillify Protocol.
